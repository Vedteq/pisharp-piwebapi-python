## Implemented

### 1. Missing Stream endpoints (values.py)
- `get_recorded_at_time(web_id, time, retrieval_mode)` ‚Üí `StreamValue`
  - `GET /streams/{webId}/recordedattime` with time and retrievalMode params
- `get_recorded_at_times(web_id, times, retrieval_mode)` ‚Üí `StreamValues`
  - `GET /streams/{webId}/recordedattimes` with repeated time params
- `get_interpolated_at_times(web_id, times)` ‚Üí `StreamValues`
  - `GET /streams/{webId}/interpolatedattimes` with repeated time params
- Both sync and async versions
- 22 new tests (happy path, query param forwarding, error paths)

### 2. Stream Updates (values.py + models.py)
- `register_stream_update(web_id)` ‚Üí `StreamUpdate`
  - `POST /streams/{webId}/updates` ‚Äî registers for change tracking
- `retrieve_stream_update(marker)` ‚Üí `StreamUpdate`
  - `GET /streams/updates/{marker}` ‚Äî fetches incremental changes
- New `StreamUpdate` model with source, latest_marker, status, events, exception
- Both sync and async versions
- 9 new tests (registration, retrieval, empty data, error paths)
- Exported `StreamUpdate` from `__init__.py`

### 3. EnumerationSet resource (enumerationsets.py + client.py)
- `get_by_web_id(web_id)` ‚Üí `EnumerationSet`
- `get_by_path(path)` ‚Üí `EnumerationSet`
- `get_values(web_id)` ‚Üí `list[EnumerationValue]`
- `get_by_data_server(data_server_web_id)` ‚Üí `list[EnumerationSet]`
- `create_value(web_id, name, value, description)` ‚Üí `None`
- Wired into both `PIWebAPIClient.enumerationsets` and `AsyncPIWebAPIClient.enumerationsets`
- 16 new tests (all CRUD operations, sync + async, error paths)
