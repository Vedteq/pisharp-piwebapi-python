### 2. Feature: `filterExpression` on `get_recorded`

- Sync + async versions
- Tests for parameter forwarding and omission (3 tests)

### 3. Feature: Element Templates resource
- New `elementtemplates.py` with `ElementTemplatesMixin` + `AsyncElementTemplatesMixin`
- Methods: `get_by_web_id`, `get_by_path`, `get_by_database`, `get_attribute_templates`
- Wired into `PIWebAPIClient` and `AsyncPIWebAPIClient` as `client.elementtemplates`
- 14 tests covering happy paths, error paths (404, 401), and parameter forwarding

### 4. Security Fix: auth_method validation and verify_ssl warning
- Reject unknown `auth_method` values with `ValueError` (prevents silent unauthenticated requests)
- Require `username` + `password` when `auth_method` is `"basic"` or `"ntlm"`
- Emit `UserWarning` when `verify_ssl=False` to alert developers
- Both sync and async clients
- 6 tests for validation and warning behavior
